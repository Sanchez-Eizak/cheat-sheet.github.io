Forms Cheatsheet

Input Tags

Tag Name Description
<form> Form Used to create an HTML form for user input
<input> Input This is the most used form element, can be changed using the "type=" attribute
<label> Label Used for accessibility, clicking on the label will toggle the button/field. "for=" attribute of <label> tag must equal "id=" attribute of the <input> element to connect.

Input Types

Type Name Description
<input type="text"> Text Field Defines a single-line input field for text input
<input type="radio"> Radio Lets a user select one of a limited number of choices
<input type="checkbox"> Checkbox lets a user select zero or more options of a limited number of choices
<input name="input-name"> Name Attribute Each input field must have a name attribute to be submitted
